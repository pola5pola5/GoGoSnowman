# GoGoSnowman

## How To Play
- →，←キーで左右に移動，spaceキーでジャンプ，enterキーで状態リセットします．
- 溶岩(lava)が落ちているので触れると溶けてしまいます．
- 空を流れる雲に触れるとsnowmanはパワーアップして大きくなります．
- 大きくなったsnowmanはlavaに触れると小さくなりますが溶けずに済みます．
- 穴を避けてゴールを目指しましょう．
